# JacaEventos
  Aplicativo de eventos criado para avalição final de Programação Coorporativa.

# Descrição do Projeto
  Neste Projeto é desenvolvida uma plataforma de eventos que permite que usuários registrados tenham a opção de participar de eventos além de também poderem cadastrar seus próprios, tomando conta da logistica do processo, pagamento, busca, localizao, etc.

** Para baixar e rodar o JacaEventos localmente: **

```
bash
$ git clone https://github.com/leonfers/JacaEventos
$ cd JacaEventos
$ pip install -r requirements.txt
```
*Basta rodar:
```
bash
$ python manage.py runserver
```
